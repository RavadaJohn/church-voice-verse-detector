
# Church Voice Verse Detector

A real-time web application that detects Bible verse references from speech and displays them automatically.

## Features

- Speech Recognition
- Bible Verse Detection
- Automatic Verse Display
- Flask Backend
- HTML/CSS Frontend
- JSON Bible Database

## Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- Git
- GitHub

## Project Goal

This project aims to help churches automatically display Bible verses on a screen whenever a pastor mentions a verse during a sermon.
